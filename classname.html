react-tabs

className

The class name to be added to the <Tabs>, <TabsList>, and <TabPanel> components.

Arguments

  1. class (String): Additional class names for the <Tabs> component.

Example

<TabList className="foo">
  <Tabs className="bar">
    <Tab>1</Tab>
  </Tabs>
  <TabPanel className="baz">One</TabPanel
</TabList>
